# imdb-movie-analytics-dashboard
An interactive Power BI dashboard built using the IMDb Movie Dataset to analyze movie ratings, revenue, genres, directors, and release trends through dynamic visualizations, KPIs, and data-driven insights.

---

# 📖 Project Overview

This project presents an interactive Power BI dashboard built using the IMDb Movie Dataset. The dashboard explores movie performance across multiple dimensions, including ratings, revenue, runtime, genres, directors, and release years.

The primary goal is to transform raw movie data into an engaging visual report that helps users discover trends, compare performance, and gain valuable insights from the entertainment industry.

---

# 🎯 Objectives

- Analyze movie ratings and audience engagement.
- Compare revenue generated by movies.
- Explore genre popularity.
- Evaluate directors based on movie performance.
- Study yearly movie release trends.
- Build an interactive dashboard for business intelligence and storytelling.
